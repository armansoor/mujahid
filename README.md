# Dr Mufti Mohammad Mujahid Ali Qasmi 
A YouTube channel based website dedicated to Dr. Mufti Mohammad Mujahid Ali Qasmi

Link: https://armansoor.github.io/mujahid/

## Development

The main source code is in `app.jsx`. After making changes to `app.jsx`, you need to compile it to `app.js` using Babel so it can run efficiently in the browser.

To do this, simply run:

```bash
npm install
npm run build
```

The website will automatically use the updated `app.js`.
